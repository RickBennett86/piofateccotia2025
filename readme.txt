#Para rodar o código:

- Instale as dependências
- Crie a pasta ".streamlit"
- Crie o arquivo "secrets.toml" e coloque a credencial de acesso ao banco e o hash.
    [database]
    url = "URL Banco"
    [credentials]
    hashed_password =  "Hash aqui"
- Rode "streamlit run app.py" ou "python -m streamlit run app.py"
- Coloque a senha disponibilizada pelo nosso grupo e acesse nosso trabalho.
